# n_bodies_heisenberg

n体ハイゼンベルク模型のハミルトニアンの行列成分を求める。

前項の、spin_operatorsから関数の定義部はコピペして良い。そしてn体のイジング模型を計算した時のように2次元配列を宣言して、処理を書いていく。

通常の行列操作ならi, jの2重ループを回す、と書いたが、今回のハイゼンベルク模型の成分を求める際は2重ループは使いづらい。
